查看 class 中的一组方法和字段，找到两个仅按大小写区别的方法和字段，这会使该类用户感到困惑。

这种情况可能只表示命名不佳。方法名称应该是面向操作的，因此包含一个动词，在方法和字段具有相同名称（有或没有大小写差异）的情况下，这是不太可能的。但是，重命名公共方法可能会对呼叫者造成干扰。因此，建议重命名成员。

不符合规范的代码示例

class SomeClass:
    lookUp = false
    def lookup():       # 不符合规范的; 方法名称与字段名称的区别仅在于大小写
        pass

合规的解决方案

class SomeClass:
    lookUp = false
    def getLookUp():
        pass
