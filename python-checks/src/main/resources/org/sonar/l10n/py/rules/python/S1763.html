跳转语句 (return, break, continue, and raise) 将控制流移出当前代码块。因此，跳转后的任何语句都是死代码。

不符合规范的代码示例

def fun(a):
  i = 10
  return i + a       # Noncompliant
  i += 1             # this is never executed

合规解决方案

def fun(a):
  i = 10
  return i + a
