根据XPath表达式的返回值创建问题。如果XPath表达式返回：

  • 单个或一组AST节点，然后为每个节点创建给定消息的行问题
  • 一个布尔值，只有当布尔值为真时才会创建给定消息的文件问题
  • 其他任何事情，没有创建问题
