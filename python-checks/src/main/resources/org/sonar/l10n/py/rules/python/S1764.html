在二元运算符的任何一侧使用相同的值几乎总是一个错误。在逻辑运算符的情况下，它是复制/粘贴错误，因此是一个错误，或者它只是浪费的代码，应该简化。在按位运算符和大多数二元数学运算符的情况下，在运算符的两边具有相同的值会产生可预测的结果，并且应该简化。

不符合规范的代码示例

if a == a: #不符合规范的
    work()

if  a != a: #不符合规范的
    work()

if  a == b and a == b: #不符合规范的
    work()

if a == b or a == b: #不符合规范的
    work()

j = 5 / 5 #不符合规范的
k = 5 - 5 #不符合规范的

  • 表达式 1 << 1
