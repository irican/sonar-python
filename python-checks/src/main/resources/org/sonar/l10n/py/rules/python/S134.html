嵌套 if, for, while, try, and with语句是制作所谓“意大利面条代码”的关键因素。 这样的代码很难阅读，重构并因此维护。

不符合规范的代码示例

以下代码段说明了此规则，默认阈值为3。

  if condition1:           # Compliant - depth = 1
    # ...
    if condition2:         # Compliant - depth = 2
      # ...
      for i in range(10):  # Compliant - depth = 3, 不超过限制
        # ...
        if condition4:     # Non-Compliant - depth = 4
          if condition5:   # Depth = 5, 超出限制，但问题仅在depth= 4时报告
            # ...
