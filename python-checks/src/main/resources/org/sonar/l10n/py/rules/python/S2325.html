不访问实例数据的类方法可以而且应该是静态的，因为它们会产生更高性能的代码。

要在Python中实现静态方法，应该使用 @classmethod 或 @staticmethod 。 类方法接收类作为隐式的第一个参数，就像实例方法接收实例一样。 静态方法不会收到隐含的第一个参数。

不符合规范的代码示例

class Utilities:
    def do_the_thing(self, arg1, arg2, ...):  # Noncompliant
        #...

合规解决方案

class Utilities:
    @classmethod
    def do_the_thing(cls, arg1, arg2, ...):
        #...

or

class Utilities:
    @staticmethod
    def do_the_thing(arg1, arg2, ...):
        #...

例外

忽略引发或可能引发 NotImplementedError 的方法。
