if / else if链，语句从上到下进行评估。 最多只执行一个分支：第一个分支的条件为 true 。

因此，复制条件会自动导致死代码。通常，这是由于复制/粘贴错误。最好的情况是，它只是死代码，最糟糕的是，它是一个可能在代码维护时引发进一步错误的错误，显然它可能导致意外行为。

不符合规范的代码示例

if param == 1:
  openWindow()
elif param == 2:
  closeWindow()
elif param == 1:            #不符合规范的
  moveWindowToTheBackground()

合规解决方案

if param == 1:
  openWindow()
elif param == 2:
  closeWindow()
elif param == 3:
  moveWindowToTheBackground()

请详见

  • CERT, MSC12-C. - 检测并删除无效或永不执行的代码
