如果声明了局部变量但未使用它，则它是死代码，应该被删除。 这样做会提高可维护性，因为开发人员不会想知道变量的用途。

不符合规范的代码示例

def hello(name):
    message = "Hello " + name # Noncompliant
    print(name)
for i in range(10):
    foo()

合规解决方案

def hello(name):
    message = "Hello " + name
    print(message)
for _ in range(10):
    foo()

例外

_ as well as tuples will not raise an issue for this rule. The following examples are compliant:

for _ in range(10):
    do_something()
username, login, password = auth
do_something_else(username, login)
